# Monad Explorer ⚡

A modern web explorer application for real-time monitoring of Monad Testnet blockchain with elegant and responsive UI.

## ✨ Features

### 📊 Real-time Network Statistics
Monitor network performance live with comprehensive metrics:
- **TPS (Transactions Per Second)** - Real-time transaction throughput
- **Block Time** - Average block creation time
- **Gas Usage** - Network gas consumption metrics

### 🔍 Block Search
Easily search blocks by block number to view complete transaction details and validator information.

### 📦 Latest Block Information
- Auto-update every 2 seconds
- Display latest blocks with complete details
- Validator information for each block

### ✅ Validator Stats and Uptime
- Monitor validator performance
- Uptime statistics based on last 20 blocks
- Comprehensive validator performance tracking

### 📜 Block History
View complete block history with:
- Validator details for each block
- Timestamp and block number
- Transaction count per block

### 💰 Staking Integration
- Wallet connect integration
- Staking management interface
- Reward tracking and claiming

### 🗺️ Validator Geolocation Map
Geographic visualization of validator locations worldwide for monitoring network distribution and decentralization.

### 📜 Address Transaction History
Powered by GoldRush API for comprehensive tracking:
- Address transaction history
- Balance history and changes
- Token transfers and interactions

### 🎨 Modern Dark Theme
- Glassmorphism design elements
- Elegant dark theme optimized for extended use
- Smooth animations and transitions

### 📱 Fully Responsive Design
Optimized for all devices:
- Desktop computers
- Tablets
- Mobile phones

## Technology Stack

Monad Explorer is built with modern technologies:
- **Next.js 14** - React framework for production
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **GoldRush API** - Blockchain data provider
