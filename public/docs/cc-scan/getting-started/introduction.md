# Introduction to CC Network Scan

## What is CC Network Scan?

CC Network Scan is an enterprise-grade blockchain explorer specifically designed for Canton Network. Built by WinSnip, it provides comprehensive monitoring, analytics, and compliance tools for enterprise blockchain operations.

## About Canton Network

Canton Network is a privacy-enabled interoperable blockchain network designed for institutional use. It enables synchronized, real-time data exchange and asset transfers across applications while maintaining privacy and compliance requirements.
