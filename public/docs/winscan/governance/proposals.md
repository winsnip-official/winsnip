# Proposals

View and understand governance proposals.

## Features
- Active proposals
- Voting status
- Proposal details
- Vote distribution
- Timeline

## Proposal States
- **Deposit Period**: Collecting deposits
- **Voting Period**: Active voting
- **Passed**: Approved by community
- **Rejected**: Not approved
- **Failed**: Didn't meet quorum

## Proposal Info
- Title and description
- Proposer
- Submit time
- Voting end time
- Yes/No/Abstain/Veto votes
- Current status

## How to View
1. Click "Proposals" in sidebar
2. Browse active and past proposals
3. Click any proposal for details
