# Introduction to WinScan

## What is WinScan?

WinScan is a comprehensive blockchain explorer and analytics platform for the Cosmos ecosystem. Built by WinSnip, it provides real-time insights into blockchain data, validator performance, network statistics, and cross-chain operations.

## Key Features

### 🔍 Block Explorer
- Real-time block and transaction tracking
- Detailed transaction information with gas analytics
- Advanced search capabilities across blocks, transactions, and addresses
- Transaction history and comprehensive analytics

### 👥 Validator Insights
- Complete validator information and performance metrics
- Real-time uptime tracking and monitoring
- Delegation management interface
- Commission rates, voting power, and governance participation

### 💰 Staking & Delegation
- Intuitive staking interface with Keplr, Leap, and Cosmostation integration
- Delegate, redelegate, and undelegate tokens seamlessly
- One-click claim staking rewards
- Advanced staking calculator for reward estimation

### 🌉 IBC & Cross-Chain
- IBC token transfers between Cosmos chains
- Channel information and connection status
- Relayer monitoring and performance tracking
- Cross-chain asset tracking and analytics

### 📊 Network Analytics
- Real-time network health monitoring
- Peer geolocation and global distribution maps
- Consensus state information and block time analytics
- Live network statistics dashboard

### 🛠️ Developer Tools
- Comprehensive REST API with full documentation
- RPC/API endpoint health checker
- State sync configuration generator
- IP and domain lookup utilities
