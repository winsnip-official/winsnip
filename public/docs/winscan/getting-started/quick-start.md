# Quick Start Guide

Get started with WinScan in just 5 minutes! This guide will walk you through the essential features and help you navigate the platform.

## Step 1: Access WinScan

Visit [https://winscan.winsnip.xyz](https://winscan.winsnip.xyz) to access the WinScan explorer.

## Step 2: Explore the Dashboard

The main dashboard provides an overview of:
- **Latest Blocks** - Real-time block updates
- **Recent Transactions** - Latest network transactions
- **Network Statistics** - TPS, block time, active validators
- **Chain Selector** - Switch between 30+ supported chains

## Step 3: Search Functionality

Use the search bar to find:
- **Blocks** - Enter block height (e.g., 12345678)
- **Transactions** - Enter transaction hash
- **Addresses** - Enter wallet address
- **Validators** - Enter validator operator address

## Step 4: Connect Your Wallet

To interact with staking and governance features:

1. Click "Connect Wallet" in the top right
2. Choose your wallet:
   - **Keplr** - Most popular Cosmos wallet
   - **Leap** - Feature-rich alternative
   - **Cosmostation** - Mobile-friendly option
3. Approve the connection request
4. Select the chain you want to interact with

## Step 5: Explore Key Features

### View Validators
- Navigate to the Validators page
- Sort by voting power, commission, or uptime
- Click on any validator for detailed information

### Check Transactions
- Go to the Transactions page
- Filter by type, status, or time range
- Click on any transaction for full details

### Monitor Network Health
- Visit the Network page
- View consensus information
- Check peer distribution on the global map

## Common Tasks

### Stake Your Tokens
1. Connect your wallet
2. Navigate to Staking → Delegate
3. Choose a validator
4. Enter amount and confirm

### Vote on Proposals
1. Go to Governance → Proposals
2. Click on an active proposal
3. Review details and vote

### Transfer IBC Tokens
1. Navigate to IBC → Transfer
2. Select source and destination chains
3. Enter amount and recipient address
4. Confirm the transfer

## Tips for Best Experience

- **Bookmark Frequently Used Chains** - Quick access to your favorites
- **Enable Notifications** - Stay updated on governance proposals
- **Use Dark Mode** - Optimized for extended viewing
- **Check Network Status** - Before making transactions
