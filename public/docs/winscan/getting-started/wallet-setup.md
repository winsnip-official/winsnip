# Wallet Setup Guide

Learn how to set up and connect your Cosmos wallet to WinScan for staking, governance, and IBC transfers.

## Supported Wallets

WinScan supports three major Cosmos wallets:

### 🦊 Keplr Wallet
The most popular wallet in the Cosmos ecosystem.

**Features:**
- Browser extension (Chrome, Firefox, Brave)
- Mobile app (iOS, Android)
- Hardware wallet support (Ledger)
- 100+ chains supported

**Installation:**
1. Visit [https://www.keplr.app/](https://www.keplr.app/)
2. Download the browser extension
3. Create a new wallet or import existing
4. Save your seed phrase securely

### 🦘 Leap Wallet
Feature-rich wallet with advanced capabilities.

**Features:**
- Browser extension
- Mobile app
- NFT support
- Built-in swap functionality
- Portfolio tracking

**Installation:**
1. Visit [https://www.leapwallet.io/](https://www.leapwallet.io/)
2. Install the extension
3. Set up your wallet
4. Backup your recovery phrase

### 🌟 Cosmostation
Mobile-first wallet with excellent UX.

**Features:**
- Mobile app (iOS, Android)
- Browser extension
- Validator services
- Governance participation
- Airdrop tracking

**Installation:**
1. Visit [https://www.cosmostation.io/](https://www.cosmostation.io/)
2. Download the app or extension
3. Create or restore wallet
4. Secure your seed phrase

## Connecting to WinScan

### Step 1: Install Your Wallet
Choose one of the supported wallets and complete the installation process.

### Step 2: Create or Import Wallet
- **New Wallet**: Generate a new seed phrase
- **Existing Wallet**: Import using your 12/24-word seed phrase

⚠️ **Important**: Never share your seed phrase with anyone!

### Step 3: Connect to WinScan
1. Visit [WinScan](https://winscan.winsnip.xyz)
2. Click "Connect Wallet" button
3. Select your wallet from the options
4. Approve the connection request
5. Choose the chain you want to use

### Step 4: Verify Connection
- Your address should appear in the top right
- Balance will be displayed
- You can now interact with staking and governance

## Security Best Practices

### Protect Your Seed Phrase
- ✅ Write it down on paper
- ✅ Store in a secure location
- ✅ Make multiple backups
- ❌ Never store digitally
- ❌ Never share with anyone
- ❌ Never enter on suspicious websites

### Use Hardware Wallets
For large amounts, consider using:
- **Ledger Nano S/X** - Most popular hardware wallet
- **Trezor** - Alternative hardware option

Connect your hardware wallet to Keplr for maximum security.

### Enable Additional Security
- Set up wallet password
- Enable biometric authentication (mobile)
- Use different wallets for different purposes
- Regularly update your wallet software

## Managing Multiple Chains

### Adding New Chains
1. Visit the chain's explorer or website
2. Click "Add to Keplr" or similar button
3. Approve the chain addition
4. The chain will appear in your wallet

### Switching Chains
1. Click on the chain selector in WinScan
2. Choose your desired chain
3. Your wallet will automatically switch

### Chain-Specific Addresses
Each Cosmos chain has a unique address prefix:
- **Cosmos Hub**: cosmos1...
- **Osmosis**: osmo1...
- **Noble**: noble1...
- **Warden**: warden1...

## Common Issues & Solutions

### Wallet Not Connecting
- Refresh the page
- Unlock your wallet
- Check if the extension is enabled
- Try a different browser

### Transaction Failing
- Ensure sufficient balance for gas fees
- Check if the chain is experiencing issues
- Verify transaction details
- Try increasing gas limit

### Chain Not Appearing
- Update your wallet to the latest version
- Manually add the chain
- Check if the chain is supported

### Balance Not Showing
- Wait for the blockchain to sync
- Refresh the page
- Check if you're on the correct chain
- Verify your address
